# Crystal Lattices and Structures

## Summary

This chapter introduces the geometric structure of crystalline solids: lattices, unit cells, and the common cubic and diamond-family lattice structures found in semiconductor materials. Students learn to describe crystal planes using Miller indices, a tool used throughout the course to discuss crystal orientation and surfaces.

## Concepts Covered

1. Crystal Lattice

3. Primitive Cell
4. Lattice Constant
5. Simple Cubic Structure
6. Body-Centered Cubic

8. Diamond Lattice Structure

11. Crystal Plane
