# Drift Current and Carrier Mobility

## Summary

This chapter introduces drift current — the current caused by an applied electric field — and diffusion current — the current caused by a carrier concentration gradient — along with carrier mobility, drift velocity, and the scattering mechanisms (lattice and impurity scattering) that limit mobility. Students use these ideas to derive conductivity, resistivity, and sheet resistance, the practical transport quantities used to characterize a doped semiconductor.
