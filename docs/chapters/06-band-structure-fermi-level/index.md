# Band Structure and the Fermi Level

## Summary

This chapter distinguishes direct and indirect band-gap materials using the E-k diagram, and introduces effective mass and density of states as tools for describing carriers within a band. Students learn the Fermi level and Fermi energy, and use band structure to classify materials as metals, insulators, semiconductors, or semimetals.
